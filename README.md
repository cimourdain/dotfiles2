# Backup
```bash
./script.sh backup
```

Then save securely the content of the `private/` folder

# Restore
## Pre-requisite
Restore manually the content of the `private/` folder


# Run
```bash
./script.sh restore
```

# Init
After restore init applications
```bash
./script.sh init
```
